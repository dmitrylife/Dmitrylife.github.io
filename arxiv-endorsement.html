Submission Context

The Psi-Continuum framework is an independent research program focused on late-time cosmology and non-equilibrium macroscopic response.

This page is part of the Psi-Continuum research framework and reflects the current stage of formal publication preparation.

The current submission is prepared for arXiv in the category hep-ph (phenomenology of high-energy physics).

The project includes:

  • peer-style structured preprints
  • fully reproducible computational pipelines
  • open datasets and likelihood implementations
  • public documentation and versioned releases

Endorsement Context

arXiv requires endorsement for submissions in certain subject categories.

The Psi-Continuum framework is presented as a documented and reproducible research program with publicly available materials, including preprints, computational pipelines, and data resources.

If you are an arXiv-endorsed researcher in a relevant category and consider the scope of this work appropriate after reviewing the materials, endorsement guidance or consideration would be sincerely appreciated.
