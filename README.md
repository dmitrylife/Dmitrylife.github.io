# Psi–Continuum

**Psi–Continuum** is an open research framework for late–time cosmic expansion,
based on macroscopic state–space diagnostics and reproducible data analysis.

At its core, the framework introduces a diagnostic coordinate:

Ψ(z) ≡ H(z) / H_ΛCDM(z) − 1

which provides a compact state–space representation of expansion history.

👉 https://psi-continuum.org/

---

## Project scope

Psi–Continuum is **not a new fundamental theory** and does not introduce
additional forces, fields, or modified gravity.

The project focuses on:

- macroscopic, background–level descriptions of cosmic expansion,
- diagnostic reconstruction of expansion histories,
- state–space representations of late–time acceleration,
- reproducible links between observational data, analysis code, and publications.
