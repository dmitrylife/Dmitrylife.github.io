# Command Line Interface (CLI)

Psi-Continuum v2 installs several useful CLI tools.

## Interactive menu

```bash
psi-cli
```

Functions:

> 1. Download datasets
> 2. Check datasets
> 3. Run full analysis pipeline
> 4. Open documentation
> 5. Show project paths

## Built-in commands

```bash
psi-download-data
```

```bash
psi-check-data
```

```bash
psi-run-all
```

Run the complete pipeline:

- model tests
- SN test
- H(z) test
- BAO tests
- joint likelihood
- ε₀ scan
- publication figures
