# Changelog

This changelog documents the development history of **Psi-Continuum v2**.

⚠️ **Archived release notice**  
Version **v0.2.3** is the **final archived release** associated with the
Psi-Continuum v2 publication.  
The codebase, API, CLI, data format, and documentation are frozen for
long-term reproducibility.

---

## v0.2.3 — Final Archived Release

- Final verification of full reproducibility pipeline
- Clean separation between software, datasets, and results
- Finalized CLI commands and entry points
- Frozen Python API (background-level only)
- Frozen documentation matching the published results
- Zenodo software archive created with DOI
- Repository marked as archived and unsupported

---

## v0.2.2 — Full Pipeline Release

- Added interactive CLI menu (`psi-cli`)
- Added dataset downloader (`psi-download-data`)
- Added automatic directory creation for results
- Added DESI DR2 BAO Gaussian likelihood
- Added joint χ² tests (SN + H(z) + BAO)
- Added ε₀ parameter scan and profiling
- Fixed all analysis scripts and unified path handling
- Added publication-ready figure generator
- Improved documentation and path resolution logic

---

## v0.2.1 — First Public Release (Not stable)

- Initial public release of Psi-Continuum v2
- Core ΛCDM and ΨCDM background solvers
- Pantheon+ SH0ES supernova likelihood
- Cosmic-chronometer H(z) likelihood
- SDSS DR12 BAO likelihood
- Basic testing and validation scripts

---

## v0.2.0 — Initial Upload

- Initial clean upload of the Psi-Continuum v2 codebase
- Internal development structure
- Early prototype analysis scripts
