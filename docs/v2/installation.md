# Installation

Psi-Continuum v2 requires:

- Python 3.10+
- NumPy ≥ 1.23
- SciPy ≥ 1.9
- Matplotlib ≥ 3.6

Install from PyPI:

```bash
pip install psi-continuum-v2
```

Or install from source:

```bash
git clone https://github.com/dmitrylife/psi-continuum-v2
cd psi-continuum-v2
pip install -e .
```
